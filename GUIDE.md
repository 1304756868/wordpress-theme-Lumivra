# Lumivra 主题安装和配置完整指南

![Lumivra](Lumivra.png)

欢迎使用 Lumivra 主题！本指南将详细介绍如何安装、配置和使用主题的所有功能。

## 安装前准备

### 系统要求检查

在安装主题之前，请确保您的服务器满足以下要求：

#### 必需条件
- ✅ WordPress 5.8 或更高版本
- ✅ PHP 7.4 或更高版本
- ✅ MySQL 5.6 或更高版本

#### 推荐配置
- 💡 PHP 8.0 或更高版本
- 💡 MySQL 8.0 或 MariaDB 10.3+
- 💡 HTTPS (SSL 证书)
- 💡 至少 256MB PHP 内存限制

### 检查 PHP 版本

1. 登录 WordPress 后台
2. 导航到 **工具 > 站点健康 > 信息**
3. 查看 **服务器** 部分的 PHP 版本

如果版本过低，请联系您的主机商升级。

### 备份现有网站

⚠️ **重要**：在更换主题前，请务必备份您的网站！

推荐备份方法：
1. 使用 **UpdraftPlus** 插件进行完整备份
2. 或使用主机商提供的备份功能
3. 备份包括：数据库 + 文件

## 主题安装

请前往 [Releases](https://github.com/ty-yqs/wordpress-theme-Lumivra/releases) 页面下载最新版本主题

### 方法一：通过 WordPress 后台安装（推荐）

1. **下载主题文件**
   - 获取 `wordpress-theme-Lumivra.zip` 文件

2. **上传主题**
   - 登录 WordPress 后台
   - 导航到 **外观 > 主题**
   - 点击 **添加新主题**
   - 点击 **上传主题** 按钮
   - 选择 `wordpress-theme-Lumivra.zip` 文件
   - 点击 **立即安装**

3. **启用主题**
   - 安装完成后点击 **启用** 按钮
   - 或返回主题页面，找到 Lumivra 主题并点击 **启用**

### 方法二：通过 FTP 安装

1. **解压主题文件**
   - 解压 `wordpress-theme-Lumivra.zip` 文件
   - 得到 `wordpress-theme-Lumivra` 文件夹

2. **上传到服务器**
   - 使用 FTP 客户端（如 FileZilla）连接服务器
   - 导航到 `/wp-content/themes/` 目录
   - 上传整个 `wordpress-theme-Lumivra` 文件夹

3. **启用主题**
   - 登录 WordPress 后台
   - 导航到 **外观 > 主题**
   - 找到 Lumivra 主题并点击 **启用**

### 方法三：通过 WP-CLI 安装

如果您熟悉命令行，可以使用 WP-CLI：

```bash
# 上传主题到服务器后
wp theme activate wordpress-theme-Lumivra

# 或直接从 zip 文件安装
wp theme install /path/to/wordpress-theme-Lumivra.zip --activate
```

## 基础配置

### 第一步：设置站点标识

启用主题后，首先设置您的网站标识：

1. **设置站点标题和标语**
   - 导航到 **设置 > 常规**
   - 填写 **站点标题**（如：我的博客）
   - 填写 **副标题**（如：记录生活，分享知识）

2. **上传 Logo**
   - 导航到 **外观 > 自定义 > 站点标识**
   - 点击 **选择 Logo**
   - 上传您的 Logo 图片
   - 推荐尺寸：400×100px（PNG 格式，透明背景）
   - 调整显示尺寸
   - 点击 **发布**

3. **设置网站图标（Favicon）**
   - 在同一页面，点击 **选择站点图标**
   - 上传图标图片
   - 推荐尺寸：512×512px
   - 点击 **发布**

### 第二步：访问主题设置

Lumivra 提供两种配置方式：

#### 方式 A：后台设置页面（推荐新手）

1. 导航到 **外观 > Lumivra 设置**
2. 您将看到一个直观的表单界面
3. 配置完成后点击 **保存设置**

**优点**：
- ✅ 传统表单，简单易懂
- ✅ 所有设置集中在一个页面
- ✅ 即时保存，无需预览

#### 方式 B：主题自定义器

1. 导航到 **外观 > 自定义**
2. 在左侧边栏选择不同的选项面板
3. 右侧实时预览更改
4. 点击 **发布** 保存更改

**优点**：
- ✅ 实时预览效果
- ✅ 可视化操作
- ✅ 更直观的体验

**优先级说明**：如果同一设置在两处都配置了，后台设置页面的配置优先级更高。

## 功能配置详解

### 1. 颜色设置

#### 主色调设置

主色调影响主题的整体风格，包括：
- 按钮颜色
- 链接颜色
- 导航菜单高亮
- 登录页面背景

**设置步骤**：

**方法 A - 后台设置页面**：
1. 访问 **外观 > Lumivra 设置**
2. 点击 **颜色设置** 选项卡
3. 点击 **主色调** 颜色选择器
4. 选择颜色或输入十六进制色值（如 `#2563eb`）
5. 点击 **保存设置**

**方法 B - 自定义器**：
1. 访问 **外观 > 自定义 > 颜色设置**
2. 调整 **主色调**
3. 实时预览效果
4. 点击 **发布**

#### 推荐配色方案

- **蓝色系**（默认）：`#2563eb` - 专业、可信赖
- **紫色系**：`#7c3aed` - 创意、优雅
- **绿色系**：`#059669` - 自然、健康
- **橙色系**：`#ea580c` - 活力、热情
- **红色系**：`#dc2626` - 激情、醒目
- **深灰色**：`#374151` - 沉稳、高端

💡 **提示**：选择与您的品牌标识相匹配的颜色。

### 2. 布局设置

#### 侧边栏位置

控制侧边栏的显示位置：

**选项**：
- **右侧**（默认）：侧边栏显示在内容右侧
- **左侧**：侧边栏显示在内容左侧
- **无侧边栏**：全宽内容，不显示侧边栏

**设置位置**：
- **后台**：外观 > Lumivra 设置 > 布局设置
- **自定义器**：外观 > 自定义 > 布局设置

**应用范围**：
- 博客首页
- 归档页面
- 搜索结果页
- 单篇文章页（可选）

💡 **提示**：如果您的内容以文字为主，推荐使用侧边栏；如果是作品展示类网站，可以选择无侧边栏。

#### 容器宽度

调整内容区域的最大宽度：

**选项范围**：960px - 1920px
**默认值**：1200px

**推荐设置**：
- **博客/新闻站**：1200px - 1400px
- **作品展示**：1400px - 1600px
- **企业官网**：1200px - 1400px

### 3. 字体设置

#### 字体系列选择

Lumivra 提供 7 种精选字体：

1. **系统默认**
   - 使用系统字体栈
   - 加载速度最快
   - 适合：所有类型网站

2. **Noto Sans SC**（思源黑体）
   - 现代无衬线字体
   - Google Fonts
   - 适合：现代设计、科技博客

3. **Noto Serif SC**（思源宋体）
   - 优雅衬线字体
   - Google Fonts
   - 适合：文学博客、传统内容

4. **LXGW WenKai**（霞鹜文楷）
   - 手写风格楷体
   - jsDelivr CDN
   - 适合：个人博客、创意内容

5. **PingFang SC**
   - 苹果设计字体
   - 系统字体
   - 适合：Apple 风格设计

6. **Microsoft YaHei**
   - 微软雅黑
   - Windows 系统字体
   - 适合：Windows 用户为主的网站

7. **Helvetica Neue**
   - 经典西文字体
   - 系统字体
   - 适合：英文网站

**设置步骤**：
1. 访问 **外观 > Lumivra 设置 > 字体设置**
2. 从下拉菜单选择字体
3. 点击 **保存设置**

⚠️ **注意**：如果选择 CDN 字体，首次访问可能需要加载字体文件。

#### 基础字体大小

调整整站的基础字体大小：

**选项范围**：14px - 20px
**默认值**：16px

**推荐设置**：
- **正文为主**：16px - 18px
- **老年用户**：18px - 20px
- **密集内容**：14px - 16px

### 4. 博客设置

#### 显示特色图片

控制文章列表是否显示特色图片：

**选项**：
- ✅ **显示**（推荐）：显示文章缩略图
- ⛔ **不显示**：纯文字列表

**特色图片规格**：
- 列表缩略图：200×140px
- 文章横幅：1200×600px
- 格式：JPEG/PNG/WebP
- 文件大小：< 500KB

#### 显示作者信息

在文章详情页显示作者信息：

**包含内容**：
- 作者头像
- 作者名称
- 作者描述
- 社交链接（如有）

**设置作者信息**：
1. 访问 **用户 > 您的个人资料**
2. 填写 **个人说明**
3. 上传 **个人资料图片**（推荐使用 Gravatar）

#### 文章摘要长度

控制文章列表中摘要的字符数：

**选项范围**：50 - 500 字符
**默认值**：150 字符

**推荐设置**：
- **简洁风格**：80 - 120 字符
- **详细预览**：150 - 200 字符

💡 **提示**：摘要显示为单行，过长的文字会被截断并显示省略号。

### 5. 页脚设置

#### 版权信息

自定义页脚的版权文本：

**默认模板**：
```
© 2026 [站点名称]. All rights reserved.
```

**自定义示例**：
```
© 2026 我的博客. 保留所有权利. | ICP备案号: 京ICP备xxxxxxxx号
```

**支持 HTML**：
```html
© 2026 <a href="https://example.com">我的博客</a>. 保留所有权利.
```

#### 返回顶部按钮

启用或禁用返回顶部按钮：

**选项**：
- ✅ **显示**（推荐）
- ⛔ **隐藏**

**按钮行为**：
- 滚动超过 300px 时显示
- 点击平滑滚动到顶部
- 固定在右下角

#### 工信部备案号

此处填写您的工信部备案号。

#### 公安备案号

此处填写您的公安备案号。

#### 公安备案跳转链接

此处填写您的公安备案跳转链接，格式为:

```txt
https://beian.mps.gov.cn/#/query/webSearch?code=xxx
```

### 6. 社交媒体链接

添加社交媒体链接到页脚：

**支持的平台**（共 8 个）：
1. 微博
2. 微信（二维码链接）
3. 豆瓣
4. 知乎
5. GitHub
6. Twitter
7. Facebook
8. Instagram

**设置步骤**：
1. 访问 **外观 > Lumivra 设置 > 社交媒体**
2. 在相应字段输入完整 URL
3. 例如：`https://weibo.com/yourname`
4. 点击 **保存设置**

**显示效果**：
- 图标自动显示在页脚
- 新窗口打开链接
- 带有 SVG 图标

💡 **提示**：只会显示已填写链接的社交媒体图标。

## 常见问题解答

### 安装相关

**Q1: 上传主题时提示"缺少 style.css 样式表"**

A: 请确保您上传的是正确的主题 zip 文件。如果您下载的文件包含多层文件夹，请解压后重新压缩正确的主题文件夹。

**Q2: 主题安装后样式混乱**

A: 可能的原因和解决方法：
1. 清除浏览器缓存（Ctrl+F5）
2. 清除 WordPress 缓存（如使用了缓存插件）
3. 检查是否有冲突的插件
4. 联系主机商检查服务器配置

**Q3: 切换主题后之前的设置丢失了吗？**

A: WordPress 的基础设置（如菜单、小工具）会保留，但主题特有的设置（颜色、布局等）需要重新配置。建议在切换主题前截图记录重要设置。

### 配置相关

**Q4: 更改了设置但前端没有变化**

A: 尝试以下步骤：
1. 点击 **保存设置** 或 **发布**
2. 清除浏览器缓存（Ctrl+Shift+Delete）
3. 清除服务器缓存
4. 检查是否启用了缓存插件（如 W3 Total Cache）

**Q5: 如何同时使用中英文字体？**

A: 选择的字体会自动处理中英文：
- 中文字体会显示中文字符
- 英文和数字会使用系统默认英文字体
- 无需额外配置

**Q6: 自定义颜色不生效**

A: 检查以下事项：
1. 颜色值格式正确（如 #2563eb）
2. 已点击保存
3. 清除缓存
4. 检查是否有自定义 CSS 覆盖

### 功能相关

**Q7: 文章缩略图不显示**

A: 检查以下事项：
1. 文章是否设置了特色图片
2. **博客设置** 中是否启用了 **显示特色图片**
3. 图片是否已成功上传
4. 如果是旧文章，安装 **Regenerate Thumbnails** 插件重新生成

**Q8: 侧边栏没有显示**

A: 可能的原因：
1. 布局设置为 **无侧边栏**
2. 侧边栏小工具区域为空
3. 使用了全宽页面模板

解决方法：访问 **外观 > 小工具**，为侧边栏添加至少一个小工具。

**Q9: 菜单下拉子菜单不工作**

A: 确保：
1. 子菜单项正确缩进
2. JavaScript 未被禁用
3. 没有插件冲突
4. 清除缓存

**Q10: 登录页面样式不正确**

A: 可能的原因：
1. 其他插件修改了登录页面
2. 浏览器兼容性问题
3. CSS 冲突

解决方法：
- 禁用其他登录相关插件
- 清除缓存
- 尝试其他浏览器

### 性能相关

**Q11: 网站加载速度慢**

A: 优化建议：
1. 使用图片压缩插件（如 Smush）
2. 启用缓存插件（如 WP Super Cache）
3. 使用 CDN 服务
4. 优化数据库
5. 选择系统默认字体（最快）

**Q12: 选择 CDN 字体后首次加载慢**

A: 这是正常现象。解决方法：

- 启用浏览器缓存

## 故障排除

### 常见错误及解决方法

#### 错误 1：白屏（WSOD）

**症状**：网站显示空白页面

**原因**：
- PHP 错误
- 内存不足
- 插件冲突

**解决方法**：
1. 启用调试模式（wp-config.php）：
   ```php
   define('WP_DEBUG', true);
   define('WP_DEBUG_LOG', true);
   ```
2. 检查 error_log 文件
3. 增加 PHP 内存限制
4. 禁用所有插件，逐个启用查找冲突

#### 错误 2：样式混乱

**症状**：页面布局错乱，无样式

**原因**：
- CSS 文件未加载
- 缓存问题
- CDN 问题

**解决方法**：
1. 清除所有缓存
2. 检查浏览器控制台错误
3. 暂时禁用 CDN
4. 检查文件权限

#### 错误 3：图片无法上传

**症状**：上传图片时报错

**原因**：
- 文件大小超限
- 权限问题
- 磁盘空间不足

**解决方法**：
1. 检查 PHP 上传限制：
   - upload_max_filesize
   - post_max_size
2. 检查 uploads 目录权限（755）
3. 联系主机商增加配额

#### 错误 4：菜单不显示

**症状**：导航菜单空白或不显示

**原因**：
- 未创建菜单
- 未分配菜单位置
- 主题文件损坏

**解决方法**：
1. 访问 **外观 > 菜单**
2. 创建菜单并添加项目
3. 勾选 **主导航菜单** 位置
4. 保存菜单
5. 如仍有问题，重新安装主题

#### 错误 5：评论不显示

**症状**：文章下方无评论区域

**原因**：
- 评论功能被关闭
- comments.php 文件问题

**解决方法**：
1. 检查文章的评论设置
2. 检查 **设置 > 讨论** 的全局设置
3. 确保 comments.php 文件存在
4. 清除缓存
