## 安装方法

### 方法一：通过 WordPress 后台安装

1. 登录到您的 WordPress 管理后台
2. 导航至 **外观 > 主题 > 添加新主题**
3. 点击 **上传主题**
4. 选择 `lumivra.zip` 文件
5. 点击 **立即安装**
6. 安装完成后点击 **启用**

### 方法二：通过 FTP 安装

1. 解压 `lumivra.zip` 文件
2. 通过 FTP 将 `lumivra` 文件夹上传到 `/wp-content/themes/` 目录
3. 登录 WordPress 后台
4. 导航至 **外观 > 主题**
5. 找到并启用 Lumivra 主题

## 系统要求

### 2. 颜色设置（两种方式均可配置）
- **主色调**：设置主题的主要颜色
- **次要色调**：设置链接和按钮的悬停颜色
- **文字颜色**：自定义正文文字颜色

#### 布局设置
- **侧边栏位置**：选择左侧、右侧或无侧边栏
- **容器宽度**：调整内容区域的最大宽度（960px - 1920px）

#### 字体设置
- **基础字体大小**：调整全站的基础字体大小
- **字体系列**：选择系统默认或衬线字体

### 2. 菜单设置

1. 导航至 **外观 > 菜单**
2. 创建新菜单或编辑现有菜单
3. 将菜单分配到以下位置：
   - **主导航菜单**：显示在页眉
   - **页脚菜单**：显示在页脚

### 3. 小工具设置

- **侧边栏**：显示在文章和归档页面的侧边

访问 **外观 > 小工具** 添加和配置小工具。

## 常见问题

### 如何更改 Logo？

1. 访问 **外观 > 自定义 > 站点标识**
2. 点击 **选择 Logo**
3. 上传您的 Logo 图片
4. 调整尺寸并点击 **发布**

### 如何设置特色图片？

1. 编辑文章或页面
2. 在右侧边栏找到 **特色图片**
3. 点击 **设置特色图片**
4. 上传或选择图片（推荐尺寸：1200×600px）

### 如何更改字体？

1. 访问 **外观 > Lumivra 设置**
2. 在 **字体设置** 选项卡中选择字体系列
3. 可选字体：
   - 系统默认
   - Noto Sans SC（思源黑体）
   - Noto Serif SC（思源宋体）
   - LXGW WenKai（霞鹜文楷）
   - PingFang SC
   - Microsoft YaHei
   - Helvetica Neue

### 侧边栏没有显示？

确保您已经在 **外观 > 小工具** 中为侧边栏添加了小工具。

### 如何禁用侧边栏？

在 **外观 > Lumivra 设置** 的 **布局设置** 中，将侧边栏位置设置为 **无侧边栏**。

### 如何自定义登录页面？

登录页面会自动使用主题的主色调和站点名称。您可以在 **外观 > Lumivra 设置** 的 **颜色设置** 中调整主色调。

### 文章缩略图显示不正确？

1. 确保已上传特色图片
2. 推荐图片尺寸：至少 200×140px
3. 如果更换主题前已有图片，可安装 "Regenerate Thumbnails" 插件重新生成缩略图

### 如何添加下拉子菜单？

1. 访问 **外观 > 菜单**
2. 将菜单项拖动到另一个菜单项下方并稍微向右缩进
3. 系统会自动显示下拉箭头和子菜单

更多问题请查看 [完整指南](GUIDE.md)。
